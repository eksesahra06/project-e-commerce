## âš™ï¸ **Instalasi & Menjalankan Project**

1. Clone repository:

   ```bash
   git clone https://github.com/eksesahra06/project-e-commerce.git
   ```
2. Masuk ke direktori project:

   ```bash
   cd project-e-commerce
   ```
3. Buat database baru di MySQL dan import file `.sql` (belum tersedia).
4. Jalankan web server lokal seperti **XAMPP** dan pindahkan folder project ke `htdocs/`.
5. Buka browser dan akses:

   ```
   http://localhost/project-e-commerce/
   ```
6. Untuk admin panel:

   ```
   http://localhost/project-e-commerce/admin/
   ```
